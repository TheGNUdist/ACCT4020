Create a Regular User

Logging in to an operating system or a database management system using the root user is an insecure practice. Instead, create a regular user using the following commands. Take a screenshot, or more than one screenshot, if necessary, of the following commands and paste the screenshot(s) into a document. Each screenshot must display your container name. Export the document to PDF:

  1. Ensure that the MySQL service is running.
  2. Log in to MySQL as the root user.
  3. Create a user: create user 'you'@'localhost' identified by 'password'; (Replace you with your first name and password with a password of your choosing.)
  4. Give your regular user permission to query your database:
    grant select on database.* to 'you'@'localhost'; (Replace database with the name of the database you created in part 1 of this project and you with the name of the user you created in the previous step.)
  5. Update privileges: flush privileges;
  6. Exit MySQL and log back in as the regular user using the same command as in step 2 but replacing root with the name of your regular user.

Run Queries

Take a screenshot of as much of the output of each query as possible, and paste all screenshots into a single document. Each screenshot must display your container name. Beneath each screenshot type the question asked and the answer that your query gave you. Export that document to PDF.

  1. Which sale approvals have no approval date?

    2. SELECT ID FROM SALE WHERE APPROVALDATE = 0000-00-00;

  2. What is the total we have paid to vendors for the current period?

    3. SELECT SUM(AMOUNT) FROM PAYMENT;

  3. How many customers does each sales rep have?

    4. SELECT SALESREP, COUNT(ID) FROM CUSTOMER GROUP BY SALESREP;

  4. Did I underpay any invoices?

    5. SELECT INVOICE.ID FROM PAYMENT JOIN INVOICE ON PAYMENT.INVOICE = INVOICE.ID
    WHERE PAYMENT.AMOUNT < INVOICE.QUANTITY * INVOICE.UNITCOST + INVOICE.FREIGHTCOST;
