Project Part 2 - Intro

This analytics project will give you practical experience using a relational database management system to analyze data. For part 1 of this project, you will install a database server, import a database, run queries, and report your results. Pretend that a client has given you access to a dataset and a Dictionary file, which you can find on the course portal, that explains the dataset. The client wants you to answer a number of questions using SQL queries. The next screen explains how to install the relational database management system MySQL. The screen screen provides a number of MySQL commands that you will need for this project. The third screen explains how to create and import the client's database. The final screen lists the questions and queries and explains how to report your results.

Queries

Take one screenshot of as much of the output of each query as possible, and paste all screenshots into a single document. Export that document to PDF. In a separate document, draft a professional email to your client that gives the answers to these questions. Export your email document to PDF, and upload both PDF files to Dropbox on eCourseware. Your grade will be based on accuracy and the professional quality of your email.

  1. Which sale approvals have no approval date?

    2. SELECT ID FROM SALE WHERE APPROVALDATE = 0000-00-00;

  2. What is the total we have paid to vendors for the current period?

    3. SELECT SUM(AMOUNT) FROM PAYMENT;

  3. How many customers does each sales rep have?

    4. SELECT SALESREP, COUNT(ID) FROM CUSTOMER GROUP BY SALESREP;

  4. Did I underpay any invoices?

    5. SELECT INVOICE.ID FROM PAYMENT JOIN INVOICE
    ON PAYMENT.INVOICE = INVOICE.ID WHERE PAYMENT.AMOUNT <
    INVOICE.QUANTITY * INVOICE.UNITCOST + INVOICE.FREIGHTCOST;
