SQL Assignment - Intro

This assignment tests your ability to query a database to find the answers to four question. You will use the Company database on the course portal, but your assignment is not to find the answers to the four questions, but rather to identify the queries necessary to find the answers. Stated otherwise, this assignment asks four questions (on the next slide). Using MySQL on your Linux container, you should determine the query that you would use to determine the answer to each question. Once you have identified the correct queries, create a document with the queries, save the document as a PDF, and upload it to Dropbox on eCourseware. I will grade solely on the accuracy of the queries.

Queries

  1. What is the first and last name of the employee with the social security number 333445555?
  2. What is the total number of hours logged in the WORKS_ON table?
  3. What is the total number of hours worked on each project by project number? (Your query should report one record for each project number.)
  4. Which employees work in the Research department? (Your query may not include "WHERE DNO = 5".)
