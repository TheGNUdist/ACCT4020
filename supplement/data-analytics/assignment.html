Practice

This assignment tests your ability to query a database to analyze data. Your assignment is not to find the answers to questions, but rather to identify the queries necessary to find the answers. Following these steps, you should identify the queries to answer the questions:

  1. Use apt to install mysql-server on your Linux container
  2. Import the Database file into MySQL
  3. Launch the MySQL client to run SQL queries

Take a screenshot of as much of the output of each query as possible, and paste each screenshot into a single document along with the query used to create the output. Export the document to PDF, and upload it to Dropbox on eCourseware. I will grade solely on the accuracy of the queries. The questions are listed on the next screen:

Questions

  1. (PAYMENT table) Which payment ID has no purchase order number?
  2. (SALE table) What is total sales revenue for the current period?
  3. (EMPLOYEE table) How many employees work at each location? (Your query should report one number for each location.)
  4. (RECEIPT and PURCHASE tables) Which, if any, receipt has a different number of raw materials delivered than were ordered in the purchase order?
  5. (SALE table) Which customer(s) placed the largest single orders?
