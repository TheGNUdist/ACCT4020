Practice

This assignment tests your ability to query a database to analyze data. Your assignment is to identify the queries necessary to find the answers.

Begin by using apt to install mysql-server on your Linux container. (You should already have installed the dialog and man-db packages previously. These are necessary to install mysql-server. During the install, leave prompts to create a password blank by simply pressing ENTER.) Then follow the steps on the next screen to create and import your database. Afterwards, use SQL to create queries to answer the questions on the third screen.

Take a screenshot of as much of the output of each query as possible, and paste each screenshot into a single document along with the query used to create the output. Export the document to PDF, and upload it to Dropbox on eCourseware. I will grade solely on the accuracy of the queries.

Instructions

  1. Log in to MySQL: mysql -uroot
  2. Create a database named company: create database company;
  3. Exit MySQL: quit;
  4. Download the database file to your container using wget and the URL of the database file on the course portal. However, between the wget command and the URL add: -O company.sql (That is a capital O, not a zero, after the hyphen.)
  5. Import the Database file into MySQL:
    mysql -uroot company < company.sql
  6. Log in to the newly created database: mysql -uroot company. From this screen, you can enter SQL queries. Use all capital letters in SQL queries.
    (e.g., SELECT ID, USERNAME FROM EMPLOYEE;)
  1. Which payment ID has no purchase order number?

    2. SELECT ID FROM PAYMENT WHERE PO IS NULL;

  2. What is total sales revenue for the current period?

    3. SELECT SUM(PRICE * QUANTITY) FROM SALE;

  3. How many employees work at each location? (Your query should report one number for each location.)

    4. SELECT LOCATION, COUNT(ID) FROM EMPLOYEE GROUP BY LOCATION;

  4. Which, if any, receipt has a different number of raw materials delivered than were ordered in the purchase order?

    5. SELECT A.ID, A.PO, A.QUANTITY - B.QUANTITY FROM RECEIPT AS A JOIN PURCHASE AS B ON A.PO = B.ID WHERE A.QUANTITY != B.QUANTITY;

  5. Which customer(s) placed the largest single orders?

    6. SELECT CUSTOMER, QUANTITY * PRICE FROM SALE WHERE (QUANTITY * PRICE) = (SELECT MAX(QUANTITY * PRICE) FROM SALE);
