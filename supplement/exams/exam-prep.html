Exam Prep - Intro

The final exam for this course will test your ability to use a Linux container to (1) install and maintain software, (2) create and manipulate files and folders, and (3) manage servers. The final exam will provide you with a series of tasks. For each task, use your Linux container to identify the correct command to complete the task. This exam prep provides mock exam tasks so that you can practice working with a Linux container in preparation for the final exam. You will not submit anything for the mock exam; it is solely for practice. If you do not know how to complete a task, you should look to the course materials for help; you should not use any other online resource (e.g., Google) because no online resource other than course materials will be available to you during the exam. This prep has 20 tasks, but the final exam will include 25, including some tasks based on the projects from this course.

Mock Exam Tasks - Page 1 of 2

  1. Create a folder named “exam-prep” (without quotes) in the current folder.
  2. Navigate to the exam-prep folder.
  3. Download the text file from “https://accountingarchitecture.github.io/exams/exam-prep” (without quotes).
  4. Print the contents of the “exam-prep” file (without quotes) to the terminal.
  5. Navigate to the parent folder of the current folder.
  6. Create a TAR archive of the exam-prep folder named “prep.tar” (without quotes).
  7. List the contents of the prep.tar archive.
  8. Update the repository list.
  9. Search for the software application with the description “advanced text-mode WWW browser” (without quotes).
  10. Install the software application from this search. (Feel free to launch it and see what the World Wide Web looks like with a console-based browser. To open the address bar, press ‘g’ (without quotes), and to quit press ‘q’ (without quotes).)

Mock Exam Tasks - Page 2 of 2

  1. Upgrade all installed applications to their newest versions.
  2. Install the mysql-server package. (Select a password for the root user during the installation.)
  3. Start the mysql service.
  4. Log in to MySQL as the root user using the password you created during installation.
  5. List all databases in MySQL.
  6. Exit MySQL.
  7. List the contents of the current folder.
  8. Delete the exam-prep folder.
  9. Navigate to your home folder.
  10. Type “I am ready!” and redirect it to a file named “FinalExam” (without quotes).
