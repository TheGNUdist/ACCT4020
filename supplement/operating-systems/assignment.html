Practice

In order to make it convenient for you to gain experience with a Linux terminal without requiring you to modify your current computer, I have selected an online service that hosts installations of Linux operating systems. Your assignment is to create a free account with Codeanywhere, add a new Blank Ubuntu 14.04 Container connection to your account, and name the container your first name and last name (e.g., ProfessorCoyne). Once you have created the account and container, turn on the container, if necessary, and open an SSH Terminal. Then perform the tasks listed on the next screen:

Tasks

  1. Use apt to update the repository list.
  2. Use apt to upgrade all installed software to newest versions.
  3. Use apt to install the package named dialog.
  4. Use apt to install the package named man-db.
  5. Use apt to search for the package name of the application “Midnight Commander” and install it (hint: not the data files or debug package). Then, uninstall the package again.

Take a screenshot after running each of these commands (If you have not learned how to take a screenshot, let me Google that for you: Windows, macOS, Chrome OS.) Paste all screenshots into a document along with the command(s) used to complete each task. Export the file to PDF and upload it to Dropbox on eCourseware.
