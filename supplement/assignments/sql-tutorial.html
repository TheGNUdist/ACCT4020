SQL Tutorial

Structured Query Language (SQL) is the primary language for interacting with relational databases, and this assignment will teach you basic SQL syntax. Complete the SQL Tutorial and SQL Quiz on W3Schools. As you complete the tutorial, you will see that it is a very simple, yet powerful language. After completing the SQL Quiz, print the results page of the quiz to PDF. You may retake the quiz as many times as you wish, but you should only submit one attempt.
