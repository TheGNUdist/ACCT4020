SQL Tutorial

This analytics project will give you practical experience using a relational database management system (RDBMS) to analyze data. For this part of the project, you will learn how to extract data from an RDBMS so that you can use that knowledge in subsequent project parts. Structured Query Language (SQL) is the primary language for interacting with relational databases. Complete the SQL Tutorial and SQL Quiz on W3Schools. After completing the SQL Quiz, print the results page of the quiz to PDF. Your grade for this part of the project will be based on your quiz score. You may retake the quiz as many times as you wish, but you should only submit your best attempt.
