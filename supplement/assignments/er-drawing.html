E-R Drawing

This part of the diagram project will give you practice drawing entity-relationship (E-R) diagrams. Knowing how to read and write E-R diagrams allows you to communicate with others about the structure of databases. The next screen displays an E-R diagram, and you should make a copy of it using the diagramming software of your choice. Draw.io is a free, online diagramming tool that you may use, but you may choose another software tool if you prefer. You can find a helpful tutorial on drawing E-R diagrams on the Draw.io website. If you use Draw.io, all required shapes will be in the Entity Relation template. When you have completed your flowchart, first, save a copy (if you use Draw.io, this will be a DRAWIO file), and second, export the flowchart as a PDF file. Upload both files to Dropbox on eCourseware.
