Interactive Assignment - Intro

For this assignment, you will compare the strength of several different passwords and passphrases using the How Secure Is My Password? tool. On the next two screens, you will find sets of passwords or passphrases. For each of the entry, write the password or passphrase in a word processor document next to the amount of time it would take to crack it. Export the document to PDF, and upload it to Dropbox on eCourseware.

Page 2 of 2 - Choose Your Own

Create your own passwords or passphrases using the rules listed below. How quickly can these be cracked?

  1. Password with 8 lowercase letters
  2. Password with 7 lowercase letters + 1 uppercase letter
  3. Password with 7 lowercase letters + 1 number
  4. Password with 7 lowercase letters + 1 special character
  5. Password with 5 lowercase letters + 1 uppercase letter + 1 number + 1 special character
  6. Password with 9 lowercase letters + 1 uppercase letter + 1 number + 1 special character
  7. Passphrase with 16 lowercase letters and no spaces
  8. Passphrase with 16 lowercase letters plus spaces
  9. Password with 16 characters in a combination of lower- and uppercase letters, numbers, and special characters
