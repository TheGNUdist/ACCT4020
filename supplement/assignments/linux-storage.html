Linux Storage

This assignment will increase your experience working with Linux by having you practice storage commands. Right click on your container's name, and select Turn On. Right click on the container name, and select SSH Terminal to access the shell. Then, complete each task listed on the next screen. For each task, identify the command necessary to complete the task. Run the command in your Linux shell, and take a screenshot after executing each command. Since the terminal window can display multiple commands, you may use a single screenshot for multiple commands, if you wish. Each screenshot must show your container name. Create a word processor document and paste each screenshot into that document. Beneath each screenshot in the document, type the task number or numbers and the Linux command used to complete each task. Export the document to PDF and upload it to Dropbox on eCourseware.

Commands

  1. Navigate to your home folder.
  2. Create a file named storage1 in the current folder.
  3. Create a folder named storage in the current folder.
  4. List the contents of the current folder.
  5. Create a file named storage2 in the storage folder without navigating to the storage folder.
  6. Print the contents of the storage folder without navigating to the storage folder.
  7. Copy the storage folder to create the storagenew folder in the current folder.
  8. Type your favorite quote and redirect it to storage1 in the current folder.
  9. Move storage1 to the storage folder.
  10. Navigate to the storage folder.
  11. Print the contents of the storage1 file to the terminal.
  12. Delete the storage2 file.
  13. Navigate to the parent folder of the current folder.
  14. Delete the storagenew folder.
