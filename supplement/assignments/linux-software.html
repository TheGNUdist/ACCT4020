Linux Software

This assignment will give you experience working with your own installation of Linux. Begin by creating an online Linux container:

  1. Create an account with Codeanywhere.
  2. Open the verification email and click on the link to verify your email address.
  3. If the Connection Wizard does not open automatically, from the main Codeanywhere screen, click File - New Connection - Container to open the Connection Wizard to create a new container.
  4. Name the container your first and last name (e.g., ProfessorCoyne).
  5. From the list of containers, select Blank - Blank Development Stack - Ubuntu 16.04, and click Create. This deploys the Linux operating system environment that you will use for the tasks on the next screens and for several other homework assignments for the remainder of this course:

Instructions

Once the container is deployed, right click on the container name and select Turn On. (Because Codeanywhere has just deployed this new container, it may already be on.) Right click on the container name, and select SSH Terminal to connect to your Linux environment's text-based shell. Using that shell, complete each task listed on the next screen. For each task, identify the command necessary to complete the task. Run the command in your Linux shell, and take a screenshot after executing each command. Each screenshot must show your container name. (If you have not learned how to take a screenshot, let me Google that for you: Windows, macOS, Chrome OS.) Create a word processor document and paste each screenshot into that document. Beneath each screenshot in the document, type the task number and the Linux command used to complete the task. Export the document to PDF and upload it to Dropbox on eCourseware.
