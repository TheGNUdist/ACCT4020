Server Assignment

Daemons or servers can seems confusing, but this assignment will demonstrate how straightforward servers can be. Using Codeanywhere, install apache2, which is a Web server. It should start automatically, but if not, launch it. On your container Info screen, which you can open by right-clicking on the container icon in the side pane and selecting Info, click the first URL, and the Apache2 Ubuntu Default Page will open in a new tab. Then, download the HTML page from the course portal to your container, extract the file from the ZIP archive, move it to the /var/www/html folder, and restart apache2. Take a screenshot of your container window after running these commands. Reload the Apache2 Ubuntu Default Page tab, and take another screenshot. Upload both images to Dropbox on eCourseware.

Line Commands

  1. sudo apt-get update
  2. sudo apt-get install apache2
  3. sudo service apache2 start (if necessary)

Pointing a web browser at this Apache install will show the default page.

  1. wget https://accounting-architecture.github.io/portal/supplement/services/index.zip
  2. unzip index.zip
  3. sudo mv index.html /var/www/html/.
  4. sudo service apache2 restart

Refreshing the web page will now show the custom page.
