Server Assignment

The concept of servers or daemons is not complicated, but it can be confusing in the beginning because users only see the hosted content (e.g., the Web site) and never visibly interact with the daemon. This assignment will demonstrate how straightforward servers can be. Using Codeanywhere, install apache2. It should start automatically, but if not, start it with systemd. From your container info screen, click the preview link, and the Apache2 Ubuntu Default Page will open in a new tab. Then, download the HTML page from the course portal, extract the file from the ZIP archive, upload the file to your container, move it to the Apache folder, and restart apache2. Reload the preview tab, and take a screenshot. If at any point the preview tab says, "This Container is currently unaccessible," then you have made a mistake.
