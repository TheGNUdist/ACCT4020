Practice

Follow these steps to set up a Web server on your Codeanywhere:

  1. Use apt to update the repository list and to install the package named apache2.
  2. Use the following command to start the Web server: sudo service apache2 start
  3. Use the following command to download the HTML File from the course portal:
    wget accounting-architecture.github.io/portal/services/index.html
  4. Move or copy index.html to the folder /var/www/html
  5. Click the Run Project button from the shortcut bar in Codeanywhere to open your Web page in a browser window.

Take a screenshot of your container window after running these commands and take another screenshot of the browser window with your Web page. Paste both screenshots into a document along with list of the commands you used to complete these tasks. Export the document to PDF, and upload it to Dropbox on eCourseware.
