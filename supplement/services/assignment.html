Server Assignment

Daemons or servers can seems confusing, but this assignment will demonstrate how straightforward servers can be. Using Codeanywhere, install apache2, which is a Web server. It should start automatically, but if not, launch it. On your container Info screen, which you can open by right-clicking on the container icon in the side pane and selecting Info, click the first URL, and the Apache2 Ubuntu Default Page will open in a new tab. Then, download the HTML page from the course portal to your container, extract the file from the ZIP archive, move it to the /var/www/html folder, and restart apache2. Take a screenshot of your container window after running these commands. Reload the Apache2 Ubuntu Default Page tab, and take another screenshot.
