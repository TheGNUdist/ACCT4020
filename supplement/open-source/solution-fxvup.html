Open Assignment

This assignment will introduce you to some open alternatives for the software you use. For the following lists of closed source software and closed formats, research and find open source alternatives for each software and each format, respectively. Create a document listing each open-closed pair, export the document to PDF, and upload it to Dropbox on eCourseware. I will grade you on whether your proposed alternatives are open.
