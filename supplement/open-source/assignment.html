Open Source Assignment

Many are unaware of open source software or feel no need to investigate it because they feel comfortable with their current software choices. This assignment moves you past this attitude. Select a software that you use, and research whether it is open source. If the software is proprietary, then research an open source alternative. Next, select a file format that you use, and research whether is an open format. If it is a proprietary format, then research an open format alternative. Finally, write a one-page single-spaced memo with proper heading explaining (1) your software and how you use it; (2) if it is proprietary, which open source alternative exists; (3) your file format and how you use it; and (4) if your format is proprietary, which open format alternative exists. Save the memo as PDF, and upload it to Dropbox on eCourseware.
