Open Source Assignment

Many individuals are either unaware of open source software or feel no need to investigate it because they feel comfortable with their current proprietary offering. This assignment circumvents both of these hurdles. Select a software that you use and research online whether it is open source. If it is open source, then complete the online quiz with the name of the open source software and leave the field for proprietary software blank. If the software is proprietary, then find an open source alternative and complete the online quiz with the names of the proprietary software and the open source alternative. Repeat this processes with a file format. If you select an open format, enter it into the quiz. If you select a proprietary format, find an open alternative and enter both into the quiz.
