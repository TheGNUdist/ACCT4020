Focus on What Matters

The police rounded up Jim, Bud and Sam yesterday, because one of them was suspected of having robbed a bank. The three suspects made the following statements under intensive questioning: Jim: I'm innocent. Bud: I'm innocent. Sam: Bud is guilty. If only one of these statements is true, who robbed the bank?

See Through the Distraction

Three statements
Jim: I'm innocent.
Bud: I'm innocent.
Sam: Bud is guilty.
Three options
Jim: TRUE
Bud: FALSE
Sam: FALSE
Jim: FALSE
Bud: TRUE
Sam: FALSE
Jim: FALSE
Bud: FALSE
Sam: TRUE

Because Bud's and Sam's statements cannot both be false, Jim's statement cannot be true. Therefore, Jim's statement is false. The other options are irrelevant. (My wife found a different solution. Can you guess?)

Be Creative

You are in a room with 100 bags of coins each with 100 coins, but only one bag has gold coins. The only difference between the 100 gold coins and the rest of the coins is that a gold coin weighs 1.01 ounce and the other coins each weigh 1 ounce. The room also has a scale, but the scale only works once. How can you identify the gold coins?

One Possible Method

Place all bags in order from 1 to 100. Take one coin from the first bag, two coins from the second, as so on until you have taken 100 coins from bag 100. Place the resulting coins—remember the induction proof to calculate how many coins that would be—on the scale. The number of ounces the scale reports above 5,050 is the number of gold coins and the corresponding bag number of the gold coins.

The Final Test

If you can, prove that difficult learning curves are steep.

I Can Disprove It

Assume that a learning curve has experience (or time) on the x-axis and learning (or knowledge) on the y-axis. Draw a steep curve and select two sections of it.

Learning increases more at A where the curve is steeper. However, a difficult learning curve results in learning increasing less which is a contradiction.
