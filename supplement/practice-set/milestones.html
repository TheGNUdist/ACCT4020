Practice Set Milestones

For five weeks prior to the practice set due date, each student will complete a portion of a practice set as a milestone. Failure to complete a milestone on time will result in a 5% penalty against your total practice set grade per milestone missed. In order to fulfill these milestone requirements, all you need to do is complete and submit the required practice set work. The practice set system generates weekly reports so that I can monitor your performance.

  • Milestone 1: Week 1 Transactions
  • Milestone 2: Week 2 Transactions
  • Milestone 3: Week 3 Transactions
  • Milestone 4: Week 4 & 5 Transactions
  • Milestone 5: End of Month Posting & Bank Reconciliation
