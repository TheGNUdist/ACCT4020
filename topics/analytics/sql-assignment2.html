SQL Assignment

  1. How many employees are listed in the EMPLOYEE table? (I include this question again because most students used an incorrect query.)
  2. Which employee's last name is last in alphabetical order?
  3. Which department has the most employees? (The query should report department name.)
  4. What is the total number of hours worked on each project by project name?
  5. Which employees are supervisors? (List first, middle, and last name. However, you may not hardcode the Ssn for your query, and the query must print each name only once.)
