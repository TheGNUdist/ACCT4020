SQL Queries

For the answers to these questions, run the following queries:

  1. How many employees are listed in the EMPLOYEE table?

    2. SELECT COUNT(Ssn) FROM EMPLOYEE;

  2. Which employee's last name is first in alphabetical order?

    3. SELECT Lname FROM EMPLOYEE ORDER BY Lname LIMIT 1; (The LIMIT 1 is optional.)

  3. What is the total number of hours logged in the WORKS_ON table?

    4. SELECT SUM(Hours) FROM WORKS_ON;

SQL Queries

  1. How many employees work in the Research department?

    2. SELECT COUNT(A.Ssn) FROM EMPLOYEE AS A JOIN DEPARTMENT AS B ON A.Dno = B.Dnumber WHERE B.Dname = 'Research';

  2. What is the total number of hours worked on each project by project name?

    3. SELECT A.Pname, SUM(B.Hours) FROM PROJECT AS A JOIN WORKS_ON AS B ON A.Pnumber = B.Pno GROUP BY A.Pname;
