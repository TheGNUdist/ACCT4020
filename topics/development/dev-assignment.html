Git and GitHub Assignment

The purpose of this assignment is to introduce you to two popular software development tools: Git and GitHub. Git is a version control system that allows developers to collaborate on projects and track the contributions of each developer. GitHub is a public online repository for hosting Git projects in the cloud. Your assignment is to contribute to a GitHub project using these steps:

Steps

  1. Create a free GitHub account. (link)
  2. Fork a GitHub repository. (link)
  3. Install Git. If you use a Container as a Service, such as Cloud9, Git may already be installed.
  4. Clone your fork of the GitHub repository.
  5. Add a text file with a quote of your choosing and your name as the title to the AddYourNameFileHere folder.
  6. Add your name and the date you completed this assignment to the list.txt file.
  7. Stage, commit, and push your changes to GitHub.
  8. Initiate a pull request to have your changes merged into the original GitHub repository.
