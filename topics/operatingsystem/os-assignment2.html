Part 2

Now that you have identified a Linux distribution or BSD flavor, you will need to learn to install it. We will use a virtual environment so that you can gain experience with the installation process without modifying your current computer setup. Because we will use this virtual machine in future homework assignments, it is important that we all have the same environment, so instead of using the distribution that you have selected, you will use the one that I recommend for new Linux users. Your assignment is download the distribution and install it in a virtual machine. After installing it, you should boot the virtual machine and take a screenshot. I have created a video tutorial to explain this process thoroughly.
