SQL Assignment

Using the Company database, answer the following questions. For each question, specify for the SQL query necessary to find the answer and the answer. You will find this assignment under the Quizzes section of eCourseware.

  1. How many employees are listed in the EMPLOYEE table?
  2. Which employee's last name is first in alphabetical order?
  3. What is the total number of hours logged in the WORKS_ON table?
  4. How many employees work in the Research department?
  5. What is the total number of hours worked on each project by project name?
