What are some examples of data validation tools?

  • Data type verifies that content has the correct data type.
  • Check digits determine that a string of digits follows allowable algorithms.
  • Presence check looks for missing values in a record.
  • Range check confirms that a value is between a predetermined minimum and maximum.
  • Uniqueness check ensures that unique values are not duplicated.
  • Look-up check limits selection to pre-existing values.
