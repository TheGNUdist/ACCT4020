November 24, 2015

News: SQL Assignment

Some students have had difficulties using MySQL or MariaDB through their Linux virtual machines. I have tried to replicate these issues, so that I could explain workarounds. First, if you install MySQL or MariaDB and never receive a password prompt during the installation, you can log in to your database without a password (i.e., mysql -uroot). The reason for this occurrence is that these password prompts require a minimum screen size that is larger than your virtual machine window. Second, if you attempt to log in to your database and receive the error: “Access denied for user 'root'@'localhost'” then you should prepend sudo to your mysql command (i.e., sudo mysql -uroot). If your error is: “Can't connect to local MySQL server through socket '/var/mysql/mysql.sock'” then you have not properly installed mariadb-server or mysql-server. If you believe that you have installed the server correctly, trying restarting your virtual machine.
